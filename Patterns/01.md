### Print the following pattern

Pattern for N = 4
```
    1
   23
  345
 4567
```
**Input Format :**

N (Total no. of rows)

**Output Format :**

Pattern in N lines

**Sample Input :**

```5```

**Sample Output :**
```
    1
   23
  345
 4567
56789
```
My output
public class Main
{
	public static void main(String[] args)
	{
		System.out.println("Pattern1");
		
		 for(int i=1; i<=5; i++) 
            {       
                int k=i;
                for(int j=1; j<=5; j++) 
                {       
                   if(j>=(6-i))
                     {
                         
                      System.out.print(k);
                       k++;
                     }
                  else
                     {
                       System.out.print(" ");
                     }
                         
                }  
                
               System.out.println( );
            } 
	}     
}
